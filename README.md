# UniGal-Toolchain

## 官方工具链 Official Toolchain

官方工具链是以“2+3+1”的形式给出。

其中，2代表两套标准(Standard)，3代表三个实现(Implementation)，1代表一份错误提案(Proposal)

### Standard

+ UniGal-Script
+ UniGal-Diagram

### Implementation

+ UniGal-Complier
+ UniGal-Render
+ UniGal-Interpreter

### Proposal

+ UniGal-EnforcementProposal
